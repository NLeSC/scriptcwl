Installation
============

* pip

    Install using pip:

    .. code-block:: sh

        pip install scriptcwl

    For development:

    .. code-block:: sh

        git clone git@github.com:NLeSC/scriptcwl.git
        cd scriptcwl
        python setup.py develop

    Run tests (including coverage) with:

   .. code-block:: sh

        python setup.py test
